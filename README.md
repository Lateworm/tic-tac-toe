# tic-tac-toe

Singleplayer Tic Tac Toe with a simple minimax AI

## Setup
```
npm install
```

### Compile for Development (with Hot Reloads)
```
npm run serve
```

### Compiles and Minify for Production
```
npm run build
```

### Run Unit Tests
```
npm run test:unit
```

### Run End-to-end Tests
```
npm run test:e2e
```

### Lint and Fix Files
```
npm run lint
```
